# 👨‍💻 Trabalhando no Projeto

Clone o Projeto:

```bash
  git clone https://link-to-project
```

Acesse o diretório que você clonou:

```bash
  cd my-project
```

Instale as dependências do projeto:

```bash
  yarn
```

Rode o projeto no localhost:

```bash
  yarn dev
```
